# Shari's Scoop

A static Jekyll site based on the [Mediumish](https://jekyllthemes.io/theme/mediumish) theme.

## Run locally

1. Install dependencies:
   ```bash
   bundle install
   ```

1. Start the Jekyll server:
   ```bash
   bundle exec jekyll serve
   ```

1. Open your browser at:
   ```
   http://127.0.0.1:4000
   ```
